## Information

Android application that lets go go over, review and order tickets for the latest movies!<br>
More information can be found [here](https://drive.google.com/file/d/1YLxeWmuEvV4cA2CKOg3NrUgiEJhuv8Ii/view).

## Features
* List new movies
* Review moves
* Watch Trailers
* Order tickets
